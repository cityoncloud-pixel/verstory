# Spec: mvp-0006 — Cloud Storage (Postgres + R2) + Auth + Responsive UI

## Goal
将 Verstory 从“本地存储”升级为“云端存储版”：预置用户登录；录音文件存 Cloudflare R2，项目/转写/改写等元数据与文本存 Postgres；UI 适配手机/平板/桌面，并将“转写中/改写中”状态放入对应板块。

## Scope
### In
- 后端：Auth（仅预置用户）、受保护 API、中间件；Postgres schema/migrations；R2 预签名 PUT/GET；录音/转写/改写相关 CRUD 与 job 状态接口。
- 前端：登录页；工作台响应式布局；录音/转写/改写三板块独立状态（含错误/进行中/完成）；跨设备刷新后可继续。
### Out
- 多用户协作/邀请、计费与配额、实时流式转写、复杂编辑器与版本对比。

## Acceptance
- 使用预置账号在 `https://story.suenbeya.com` 登录后能创建项目、录音并上传到 R2，并在另一设备登录后看到并能播放同一录音。
- 触发转写/改写时，“转写中/改写中”状态显示在各自板块标题栏；失败只影响对应板块。
- `npm run dev` 可跑通一次端到端流程；`npm run build` 成功。
