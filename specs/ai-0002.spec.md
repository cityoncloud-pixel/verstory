# Spec: ai-0002 — STT transcribe API

## Goal
实现 `POST /api/stt/transcribe`：上传音频并返回转写文本，支持 provider/model 选择与清晰错误码。

## API
- `POST /api/stt/transcribe` (multipart/form-data)
  - fields:
    - `audio` (file)
    - `provider` (optional; default from registry)
    - `model` (optional; default from registry)
    - `language` (optional; e.g. `zh`)
  - response:
    - ok: `{ ok:true, text, provider, model, duration }`
    - error: `{ ok:false, error:{ code, message, details? } }`

## Constraints
- 文件大小限制：<= 25MB
- 无 API key 时：返回 `MISSING_API_KEY`（401）

## Acceptance
- 无 key：上传任意文件时返回 `401` 且 `error.code=MISSING_API_KEY`
- 有 key：上传可识别音频后返回 `ok:true` 与 `text`
