# Spec: mvp-0004 — UI Flow + Export Markdown

## Goal
完成端到端可演示闭环：在同一页面完成录音→转写（mock）→生成故事（mock）→导出 Markdown，并确保按钮状态与提示清晰。

## Scope
### In
- 导出 Markdown：
  - 下载 `.md` 文件（必做）
  - （可选）复制到剪贴板（若浏览器支持）
- UI 细节：
  - “导出”在无故事文本时禁用并提示
  - 统一状态/错误提示展示

### Out
- 富文本编辑器/模板系统
- PDF/Docx 导出

## Acceptance / Verification
- 手工：生成故事后点击导出，下载到本地并内容正确（包含标题与时间戳）
- build：`npm run build` 通过
