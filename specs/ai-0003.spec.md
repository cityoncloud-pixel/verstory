# Spec: ai-0003 — Text refine API + prompts

## Goal
实现 `POST /api/text/refine`：对输入文本进行清理/整理/目标化（GAEH goal packet），并支持 provider/model 选择与清晰错误码。

## API
- `POST /api/text/refine`
  - body: `{ text, mode: clean|organize|goal, provider?, model? }`
  - response:
    - ok: `{ ok:true, result, mode, provider, model, duration }`
    - error: `{ ok:false, error:{ code, message, details? } }`

## Acceptance
- 无 key：返回 `401` 且 `error.code=MISSING_API_KEY`
- mode=clean/organize/goal 都能返回非空 result（有 key 时）
- goal 模式输出符合 goal packet 模板（含各段标题）
