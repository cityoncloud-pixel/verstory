# Spec: ai-0004 — Frontend integration (model select + upload + refine)

## Goal
前端接入 Phase 4 后端 API，实现可配置模型的端到端闭环：
录音（或选择片段）→ STT（API）→ 文本整理（API，mode 可选）→ 展示与导出。

## Acceptance
- 前端可获取模型列表（/api/models）并渲染 provider/model 下拉框
- 用户可选定一个录音片段并点击“转写（API）”
- 用户可选择 refine mode 并点击“整理（API）”
- 无 key 时前端显示后端返回的 `error.code`（例如 MISSING_API_KEY）
