# Spec: mvp-0001 — Scaffold Web App (Vite + React + TS)

## Goal
在仓库内新增一个可运行、可构建的前端工程，为后续录音/存储/provider/UI 串联提供落点。

## Scope
### In
- 新增 `apps/web` 前端工程（Vite + React + TypeScript）
- 单页骨架（不做复杂路由）：项目区 + 录音区 + 文本区 + 导出区（可先占位）
- 最小状态：可创建/选择项目（先内存态或 localStorage；IndexedDB 留待 `mvp-0002`）
- 基础样式与可读布局（不追求精美 UI）

### Out
- 真实录音/存储/provider 实现（留给后续任务）
- 账号体系与后端服务

## Acceptance / Verification
- `cd apps/web` 后可 `npm install`
- `npm run dev` 启动后可打开页面，看到四块区域与“创建项目/选择项目”最小交互
- `npm run build` 成功

## Notes
- 工程目录选择 `apps/web` 是工程实现细节决策，用于与 GAEH harness 目录隔离。
