# Spec: ai-0001 — Model registry + /api/models + /api/config/check

## Goal
让系统支持“可选 provider/model”，并提供可验证的模型列表与配置检查接口。

## Endpoints
- `GET /api/models`
  - 返回 providers、models、defaults（不回传密钥）
- `GET /api/config/check`
  - 返回每个 provider 的 enabled 与 apiKey 是否就绪（缺失给出缺失 env 名称）

## Provider Registry (MVP)
- openai
  - stt: `whisper-1`, `gpt-4o-mini-transcribe`
  - text: `gpt-4.1-mini`, `gpt-4o-mini`
- deepseek
  - text: `deepseek-chat`, `deepseek-reasoner`

## Acceptance / Verification
- 无密钥时：`/api/models` 仍返回列表；`/api/config/check` 明确指出缺失
- 有密钥时：`/api/config/check` 显示 ready
