# Spec: mvp-0002 — Recording + IndexedDB Storage (Audio Segments)

## Goal
实现可验收的分段录音与本地持久化：在同一项目下录多段音频，保存到 IndexedDB，刷新页面后仍可恢复并回放。

## Scope
### In
- 录音：浏览器 `MediaRecorder`，支持开始/停止/继续（继续 = 录制新片段追加）
- 持久化：IndexedDB 存储 Project 与 RecordingSegment（含 blob）
- UI：片段列表展示（创建时间/时长/回放/删除）
- 恢复：刷新页面后可加载项目列表与片段列表

### Out
- 转写与故事整理（留给 `mvp-0003` 及以后）
- 云端同步、账号

## Data Schema (IndexedDB v1)
- `projects`（keyPath: `id`）
  - `{ id, name, createdAt, updatedAt }`
- `segments`（keyPath: `id`，index: `projectId`）
  - `{ id, projectId, createdAt, durationMs, mimeType, blob }`

## Acceptance / Verification
- 手工验收：
  1) 新建项目并选择
  2) 录两段：开始→停止→继续→停止
  3) 列表出现两段，可回放
  4) 刷新页面：仍可看到项目与两段音频，并可回放
- 回滚安全：本任务仅新增/改动前端代码与本地存储结构，不做破坏性迁移（v1 新建）
