# Idea

本项目旨在开发一个网页应用，用于支持用户进行口述自传式的故事记录与整理。用户可以在网页中创建一个项目，点击录音按钮开始录音，录音结束后可以保存当前录音，并支持继续录音、分段补充内容。系统会将用户的录音内容转写为文字，并调用 LLM 对转写文本进行语法、字词、表达逻辑和叙事结构的整理，最终生成一篇通顺、连贯、接近自传口吻的故事文本。

文本整理的核心原则是尊重口述人的原意。LLM 不应过度改写、虚构或改变叙述内容，只在必要时对语病、重复、断句、口头语、逻辑顺序和表达不清之处进行适度修正，使内容更易读、更完整、更像一篇自然流畅的口述故事。

项目初期将在本机环境中完成测试和验证，后续需要支持在平板设备上运行。用户界面应简洁清晰，至少包含项目创建、开始录音、结束录音、继续录音、录音转文字、文本整理生成故事、继续补充并再次转化等功能。用户可以在页面下方的文本框中查看录音转写后的原始文字，也可以查看经过 LLM 整理后的故事文本。

本项目的基本功能要求包括：能够录音、能够续录、能够将录音转写为文字、能够调用 LLM 将文字整理为故事、能够在已有内容基础上继续转化，并且整个流程应保持用户输入内容的连续性和一致性。
