# Plan: mvp-0006 — Cloud Storage (Postgres + R2) + Auth + Responsive UI

- Add Postgres schema + migrations for: users, projects, recordings, transcripts, rewrites, jobs, refresh_tokens
- Add auth: login (pre-provisioned users), access token, refresh cookie, auth middleware
- Update CORS for `https://story.suenbeya.com` with credentials support

## 2) R2 storage
- Add R2 client (S3-compatible) using env vars
- Implement recording upload flow:
  - `POST /api/projects/:id/recordings/init` (returns recordingId + presigned PUT)
  - Browser `PUT` to R2
  - `POST /api/recordings/:id/complete` (persist metadata)
- Implement playback flow:
  - `GET /api/recordings/:id/signed-url` (returns short-lived GET URL) or stream proxy

## 3) STT & rewrite job wiring

- Ensure “转写中/改写中” can be driven by per-job status (polling endpoints)
